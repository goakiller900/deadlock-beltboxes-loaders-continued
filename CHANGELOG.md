# Changelog

## 2.7.0

Initial community-maintained continuation release for Factorio 2.1.

### Changed

- Updated mod metadata for Factorio 2.1.
- Updated the project title and homepage to point to the continued project.
- Raised the base dependency to `base >= 2.1`.

### Fixed

- Added compatibility for the Factorio 2.1 `RecipePrototype` change where `category` and `additional_categories` were merged into `categories`.
- Verified basic in-game functionality with loaders, beltboxes, stacking, and unstacking.

### Notes

- This is an unofficial community-maintained continuation of the original mod.
- Original authors remain credited in `info.json` and the README.
- The original GPLv3 license remains in effect.
